# AI 操作记录目录

本目录保存 AI 对 vault 执行整理、迁移、检查和规则更新时产生的操作记录。

## 规则

- 操作记录使用中文。
- 与用户相关的沟通摘要、确认事项和结果说明使用中文。

- 每次操作新建一个文件，不修改旧记录。
- 操作记录不使用 Obsidian 双链，使用普通相对路径。
- 不同操作类型分目录保存，不混在一个持续增长的总日志里。
- 除非用户明确要求修复事实错误或编码损坏，否则不回改历史操作记录。

## 子目录

```text
rule-updates/   规则更新记录
reviews/        标题、位置、链接、重复、孤岛页检查报告
migrations/     文件或目录迁移计划和迁移结果
maintenance/    其他维护操作
external/       vault 外部文件操作（系统配置、工具安装等）
legacy/         旧日志，只保留追溯，不继续写入
```

## 文件命名

```text
YYYY-MM-DD_HHMM_操作摘要.md
```

示例：

```text
2026-06-26_2111_rule-log-folder-migration.md
```
