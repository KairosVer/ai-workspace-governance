# Templates

This directory contains reusable workspace templates.

## generic-workspace

A tool-agnostic template for AI-managed file workspaces.

Use it for:

- research folders
- writing projects
- documentation repositories
- team shared workspaces
- AI-assisted project archives

## Adapters

Tool-specific templates live in `adapters/`.

Current adapter:

```text
adapters/obsidian-vault/
```
