# Templates

This directory contains reusable workspace templates.

## generic-workspace

A tool-agnostic template for AI-managed file workspaces.

Use it for:

- research folders
- writing projects
- documentation repositories
- team shared workspaces
- AI-assisted project archives

## Adapters

Tool-specific templates live in `adapters/`.

Current adapter:

```text
adapters/obsidian-vault/
```

## Navigation Files

Templates may include a small navigation layer:

- `00_System/00-Home.md` for generic workspaces.
- `8-Atlas/00-Home.md` for Obsidian vaults.
- `00_Guide/` or `00-Guide/` folders for local folder guides.

These files are intentionally short. They are entry points, not complete inventories.
