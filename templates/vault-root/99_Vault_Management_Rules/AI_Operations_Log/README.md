# AI 操作记录

本目录保存 AI 对 vault 进行整理、迁移、检查、规则更新时产生的单次操作记录。

## 子目录

```text
rule-updates/  规则新增、修改、拆分、合并、废弃
reviews/       标题、位置、链接、重复内容、孤岛页面等检查报告
migrations/    文件移动、目录重组、批量重命名
maintenance/   小规模维护，如创建索引、补模板、修链接
legacy/        旧日志或旧格式记录
```

## 规则

- 每次操作新建一篇记录。
- 不向单一总日志追加。
- 不反复修改旧记录。
- 使用普通相对路径，不使用 Obsidian 双链。
- 不写入个人隐私、账号、密钥或不必要的本机绝对路径。
