# Safety Rules

These are the highest-priority rules for this workspace.

## Human Confirmation Required

Before any of the following, list affected paths, explain risk, and wait for confirmation:

- deleting files or folders
- recursive deletion
- moving more than 3 files
- renaming more than 3 files
- overwriting user-authored content
- changing generated output that may be final
- publishing or exporting private material
- editing files with uncertain encoding

## Prefer Archive Over Delete

When uncertain, move material to `05_Archive` instead of deleting it.

## Batch Move Checklist

Before moving files, list:

- source path
- target path
- number of files
- overwrite risk
- link/reference risk
- rollback plan

## Privacy

Do not publish or copy into public docs:

- personal notes
- private paths
- credentials
- API keys
- internal project names
- unreleased research or business material

Use fictional examples and relative paths.

## AI Must Not Fabricate

AI must not invent sources, data, decisions, approvals, or file history.

When uncertain, mark uncertainty explicitly.
