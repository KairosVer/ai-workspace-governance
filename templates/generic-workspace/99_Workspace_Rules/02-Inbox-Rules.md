# Inbox Rules

`01_Inbox` is for low-friction capture.

It may contain raw material, temporary ideas, imports, AI transcripts, and files whose final location is unclear.

Rules:

- Preserve original context.
- Do not rewrite raw records into polished conclusions.
- Add source information when available.
- Move material out only after its purpose is clear.

Routing:

- reusable insight -> `02_Knowledge`
- active work -> `03_Projects`
- deliverable draft -> `04_Outputs`
- inactive but worth keeping -> `05_Archive`
