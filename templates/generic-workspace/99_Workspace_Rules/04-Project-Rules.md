# Project Rules

`03_Projects` stores active work.

A project should make it easy to answer:

- What is the goal?
- What is the current state?
- What is the next step?
- What decisions have been made?
- What are the risks?
- Where are the related inputs, knowledge, and outputs?

Suggested project files:

```text
index.md
progress.md
tasks.md
decisions.md
notes.md
```

Small projects may only need `index.md`.
